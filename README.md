> App de escritorio para la impresión de recetas sistema meppes y ambulatorio.

### 🏠 [Homepage](https://github.com/electron-react-boilerplate/electron-react-boilerplate#readme)

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev
```

## Run tests

```sh
yarn test
```

## Run build

```sh
yarn package
```
